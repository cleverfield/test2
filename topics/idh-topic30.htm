Installation

 

 

 

 

Precept is installed using Microsoft ClickOnce technology.ClickOnce is a deployment technology that enables to create self-updating Windows-based applications that can be installed and run with minimal user interaction.

 

More information on ClickOnce Technology can be found in Microsoft website:

https://learn.microsoft.com/en-us/visualstudio/deployment/clickonce-security-and-deployment?view=vs-2022

 

 

Usually, Precept is installed using the following link, unless a different installation procedure were required by your company

 

https://precept.blob.core.windows.net/install/publish.htm

 

 

 

 

Installation will start after clicking on the Install button.

 

Microsoft ClickOnce allows rapid bug fixes since after a bug correction, Precept will automatically update itself during the next launch.

 

 

Precept can be launched from the desktop by clicking on its shortcut icons.

Users are then required to enter their Windows ID (same as used to login in their corporate account) and password (provided by Cleverfield)

 

 

Precept uses a per-named license system, as opposed to concurrent licenses, meaning a license has to be purchased for every users.
