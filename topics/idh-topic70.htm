Network modelling

 

 

 

 

Network modelling and its associated objective function to solve and optimize production needs to be performed with great care to ensure accurate results and fast computation time. Don't hesitate to ask for Cleverfield help if ensure how to model your asset.

 

The network can be visualized and edit by clicking on the "Network" icon from Precept main User Interface.

 

The network is displayed on the right on the screen. Constraints can be given at any node of the network using the table on the left. Those are initial constraints at start of forecast. Constraints can be modified with time (see section on Constriants chagnes vs time).

 

The first column "Rank" has been introduced for visualization purposes. In the Timeline in the main Precept interface, nodes and well are ranked by Rank Ascending first, then alphabetically. This feature is useful to have the nodes and wells ranked as per users preference.

 

A node or well with a rank greater than 1000 will not be shown in the timeline in the main Precept interface.

 

 

 

 

Network items can be added using existing items using Ctrl+C /Ctrl+V functions.

 

New items can also be added by clicking on the "Draw network" icon, and then drag them into the production network.

 

 

A color code is used for pipeline to indicates the type of  fluids flowing through.

 

Reporting nodes and pipeline

Reporting nodes and pipelines do not physically exist but can be used to sum fluids from various nodes. In such case, the fraction of fluid coming from the 'parent-node" can be specified by clicking on the list box and select "view reporting Node". This feature can be used to calculate entitled production in case a field is produced under a production sharing agreement between several companies.

 

Split Nodes

Fluids from a parent-node can be sent to more than one child-node.  if no specific split fraction is introduced, Precept solver will split the fluids in order to maximize production. However a split fraction can be introduced by clicking on the list box and select "view Splits".
